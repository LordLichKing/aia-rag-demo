# 员工手册

## 1. 公司简介

本公司是一家专注于人工智能技术研发与应用的科技企业，致力于为全球客户提供智能化解决方案。公司秉承"创新、协作、诚信、卓越"的核心价值观，持续推动技术进步和业务发展。

## 2. 假期管理

### 2.1 年假政策

员工根据工龄享有不同天数的年假：
- 工龄1-5年：每年10天年假
- 工龄6-10年：每年15天年假
- 工龄11年以上：每年20天年假

年假应在当年内使用完毕，如因工作原因未能休完，经部门主管批准可延期至次年第一季度。

### 2.2 病假

员工每年享有10天带薪病假。连续病假超过3天需提供医院证明。

### 2.3 事假

事假需提前申请，经部门主管批准后方可休假。事假期间不计薪酬。

## 3. 加班管理

### 3.1 加班申请

员工加班需提前通过OA系统提交加班申请，经部门主管批准后方可加班。紧急情况可先加班后补申请，但需在加班后2个工作日内完成补批。

### 3.2 加班补偿

- 工作日加班：按1.5倍工资计算
- 周末加班：按2倍工资计算或安排调休
- 法定节假日加班：按3倍工资计算

## 4. 报销流程

### 4.1 报销申请

员工需在公司财务系统中提交报销申请，附上相关发票和凭证。报销类别包括：
- 差旅费
- 招待费
- 办公用品费
- 培训费

### 4.2 审批流程

1. 员工提交报销申请
2. 部门主管审批（金额≤5000元）
3. 财务部门审核
4. 财务总监审批（金额>5000元）
5. 报销款项在审批通过后5个工作日内到账

## 5. 保密协议

所有员工入职时需签署保密协议，不得泄露公司商业秘密、技术资料和客户信息。离职后保密义务持续2年。

## 6. 行为准则

- 遵守法律法规和公司规章制度
- 诚实守信，不得弄虚作假
- 尊重同事，营造良好的工作氛围
- 保护公司财产和信息安全
